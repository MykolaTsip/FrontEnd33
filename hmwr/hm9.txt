* Кожне завдання відображайте на сторінці роздільно.

1. Позиціонування:
Створіть сторінку з декількома блоками елементів.
Використовуйте різні значення властивості position (relative, absolute, fixed) для цих блоків.
Використовуйте z-index для управління відображенням блоків на сторінці.

2. Таблиці:
Створіть HTML-таблицю з даними, такою як розклад тижня чи список товарів.
Використовуйте елементи <thead>, <tbody>, та <tfoot> для структури таблиці.
Застосуйте стилі CSS для призначення кольорів та вирівнювання тексту.

3. Відступи:
Створіть контейнер і вкладіть в нього декілька блоків.
Використовуйте властивості margin та padding для вирівнювання та відступів між блоками.

4. Лінійні та блокові теги:
Створіть список з лінійними (<span>) та блоковими (<div>) елементами.
Застосуйте стилі, які роблять лінійні елементи іншого кольору чи фону, щоб вони виділялися.

5. Медіа:
Додайте на сторінку зображення, відео та аудіо елементи.
Застосуйте атрибути width та height для зображень.
Використовуйте тег <video> для відтворення відео та <audio> для відтворення аудіо.

6. Гіперпосилання:
Створіть меню навігації на сторінці за допомогою гіперпосилань (<a>).
Використовуйте атрибут href для посилань на різні розділи вашого сайту.
Застосуйте стилі для виділення активного посилання.

7. Текст:
Використовуйте властивості CSS, такі як font-family, color, та background-color для стилізації тексту та фону.
Використовуйте властивості border для надання рамок блокам або таблицям.
Не забувайте валідність HTML та CSS.
